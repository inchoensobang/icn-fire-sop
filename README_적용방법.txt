적용 방법

1. 이 zip을 풀어서 나온 4개 파일을 GitHub Desktop 로컬 저장소 폴더(icn-fire-sop)에 넣습니다.
   - index.html
   - style.css
   - script.js
   - sopData.js

2. 기존 index.html은 덮어쓰기 합니다.

3. PDF 폴더 구조는 그대로 둡니다.
   예:
   PDF/SOP208_소방호스연장표준작전절차/1.jpg
   PDF/SOP208_소방호스연장표준작전절차/2.jpg
   PDF/SOP2026.pdf

4. GitHub Desktop에서
   Summary: Ver 2.0 SOP 이미지 뷰어 적용
   Commit to main
   Push origin

5. 사이트 접속 후 테스트:
   https://inchoensobang.github.io/icn-fire-sop/

주의:
- 폴더명은 sopData.js의 folder 값과 정확히 같아야 합니다.
- 각 SOP 폴더 안 이미지는 1.jpg, 2.jpg, 3.jpg ... 순서여야 합니다.
- 전체 원본은 PDF/SOP2026.pdf 이어야 합니다.
